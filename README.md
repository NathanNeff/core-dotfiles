# core-dotfiles

Minimal dotfiles necessary for presentation on VMs

## Installation

* Vim

```bash    
    export DOTFILES_DIR=~/src/core-dotfiles
    ln -s $DOTFILES_DIR/vim/vimrc ~/.vimrc
```

* Tmux

```bash
    export DOTFILES_DIR=~/src/core-dotfiles
    ln -s $DOTFILES_DIR/tmux.conf ~/.tmux.conf
```
